# 🌿 Greenden

Greenden is a modern, responsive plant shop landing page built using **HTML** and **Tailwind CSS**. The project features a clean and user-friendly interface designed to showcase indoor plants, gardening products, and services with an attractive layout.

## 🚀 Features

- 🌱 Responsive design for all screen sizes
- 🎨 Modern UI built with Tailwind CSS
- 🛒 Product showcase section
- 💬 Customer testimonials
- 📩 Newsletter subscription section
- 📱 Mobile-friendly navigation
- ⚡ Fast and lightweight

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
